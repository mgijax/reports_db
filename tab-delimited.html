MGI Tab-Delimited Reports

Tab-Delimited Data Files
  1. Markers
    • MGI Acc ID, marker symbol
    • MGI Acc ID, marker symbol, name, cM position, chromosome

  2. Nucleotide Sequence Accession IDs
    • MGI Acc ID, marker symbol, name, cM position, chromosome, list of space-separated Nucleotide Sequence Accession IDs, UniGene Accession ID

  3. SwissProt Protein Sequence Accession IDs
    • MGI Acc ID, marker symbol, name, cM position, chromosome, list of space-separated SwissProt Protein Sequence Accession IDs

  4. Mapped ESTs
    • EST MGI Acc ID, EST GenBank ID, chromosome, cM position, marker symbol, marker name

  5. Mouse/Human Homology
    • MGI Acc ID, marker symbol, name, GDB Acc ID, GDB marker symbol

  6. Gene Ontology Associations
    • MGI Acc ID, Gene Symbol, GO ID, ...
      (for a complete file format description see GO Annotation File Format)
    • Gene Ontology, GO ID, GO Term

  7. Strains
    • MGI Acc ID, Strain, Gene(s) corresponding to Allele(s) carried on Strain, External Accession IDs, Synonyms
